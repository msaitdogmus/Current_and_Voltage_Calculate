# Current and Voltage Calculate

A Python desktop application for analyzing oscilloscope-based **current** and **voltage** waveform data, visualizing results, and exporting PDF reports.

This repository includes two standalone desktop tools:

- **Current.py** for current waveform analysis
- **Voltage.py** for voltage waveform analysis

Both applications provide CSV-based analysis, graphical visualization, configurable scaling, and PDF report generation.

---

## Features

- Analyze waveform data from CSV files
- Separate tools for **current** and **voltage** calculations
- Interactive desktop GUI built with **Tkinter**
- Graph plotting with **Matplotlib**
- PDF report generation with **ReportLab**
- Optional drag-and-drop CSV support
- Configurable scaling factor
- Configurable application title, organization name, and authentication settings
- Automatic PDF opening after export
- English one-line comments added for function readability
